# Atas - Padrao de Registro

Data de criacao: 2026-04-20

Esta pasta guarda atas de reuniao tecnica/produto com decisoes que impactam roadmap, escopo, riscos e alinhamentos operacionais.

## 1. Quando registrar uma ata

- mudanca de escopo aprovada;
- decisao arquitetural com impacto em mais de um modulo;
- alteracao de prioridade de backlog;
- definicao de criterio de aceite entre produto e engenharia.

## 2. Convencao de nome de arquivo

Padrao:
- `YYYY-MM-DD-ata-<tema-curto>.md`

Exemplo:
- `2026-04-20-ata-flashcards-bloco-4-planejamento.md`

## 3. Template minimo

```md
# Ata - <tema>

Data:
Participantes:
Objetivo:

## Decisoes tomadas
- ...

## Pendencias
- ...

## Responsaveis e prazos
- ...

## Impacto no sistema
- ...
```

## 4. Observacoes

- manter linguagem objetiva e orientada a decisao.
- evitar texto narrativo longo sem acao.
- referenciar PRs, tarefas e docs impactados quando houver.
